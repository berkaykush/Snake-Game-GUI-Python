## Packages To Install:

```
pip install pygame
```

## Running the Snake:

```
git clone https://github.com/berkaykush/Snake-Game-GUI-Python.git
cd Sanke-Game-GUI-Python
python code/main.py
```
or 
```
git clone https://github.com/berkaykush/Snake-Game-GUI-Python.git
cd Sanke-Game-GUI-Python/code
python main.py
```
